Interfaces y lenguajes de programación

Definición general

Crearemos un sitio web, un espacio digital en internet, donde albergará diversos documentos como, texto, fotos, videos,etc. que se estructuran para dar presencia en línea a una marca, empresa, negocio, producto o servicio. También se puede definir como conjunto de páginas web bien organizadas, a la vez es un documento electrónico que puede estar creado con código HTML, se necesita un alojamiento llamado hosting y se relaciona a un dominio que le permite ser visibles en internet por cualquier acceso.

HTML es el lenguaje de marcas de hypertexto, creado en 1991 por Tim Berners-Lee y que contenia originalmente 22 elementos, la versión definitiva de HTML5 se publicó en Octubre de 2014. El CSS, es el código que da estilo al contenido web, no es un lenguaje de programación, lo usas para diseñar elementos HTML de forma selectiva.
